## Cara Instalasi

1. Copy folder aplikasi PerhitunganLembur ke lokal
2. Atur database pada .env
3. Buat database sesuai yang sudah diatur di file .env
4. Migrasi database dengan menjalankan `php artisan migrate`, lalu jalankan seeder dengan `php artisan db:seed DatabaseSeeder` di command prompt atau terminal
5. Jalankan web server Laravel dengan jalankan perintah `php artisan serve` di command prompt atau terminal
6. Untuk melakukan testing, jalankan perintah `vendor/bin/phpunit` pada folder utama aplikasi di command prompt atau terminal
7. Untuk mengakses dokumentasi, jalankan perintah `php artisan l5-swagger:generate` pada terminal, lalu akses pada `http://http://localhost:8000/api/documentation#/default`
